# Gestion des Contacts - CRUD Simple

## Installation

### 1. Importer la base de données
```bash
mysql -u root -p crud_users_db < crud/crud_users_db.sql
```

### 2. Accéder à l'application
```
http://localhost/developpement_web_2_web_dynamique/crud/read.php
```

## Fonctionnalités

- ✅ Ajouter un contact (create.php)
- ✅ Afficher la liste (read.php)
- ✅ Modifier un contact (update.php)
- ✅ Supprimer un contact (delete.php)

## Fichiers

- `crud/db.php` - Connexion à la base de données
- `crud/read.php` - Liste des contacts
- `crud/create.php` - Ajouter un contact
- `crud/update.php` - Modifier un contact
- `crud/delete.php` - Supprimer un contact
